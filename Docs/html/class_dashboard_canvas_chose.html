Detailed Description

Clase que insertara los datos de nombre y puntuacion en la base de datos ademas de mostrar el nombre del jugador por pantalla.

Tambien se usara como recursos para funciones que se llamaran cuando el usuario introduzca su nombre.

Parameters
canvasInputCanvas que mostrara el campo input donde el jugador tendra que indicar su nombre
canvasPuntuacionCanvas que mostrara la puntuacion de la partida que se acaba de terminar ademas de las top 3 puntuacion historicas junto al nombre del jugador que la hizo
nombreCampo donde el usuario indicara su nombre para posteriormente ser mostrado junto a su puntuacion
nombreGuardadoNombre que guardaremos para que se introduzca en la base de datos

◆ terminadoInput()

void DashboardCanvasChose.terminadoInput ( )

Funcion que se ejecutara cuando el jugador haya terminado el input de su nombre.

Cogeremos el nombre introducido y lo pondremos en el nombreDashboard para que se vea por pantalla, ademas lo guardaremos en la variable nombreGuardado para posteriormente introducirlo en la base de datos. Tambien desactivaremos el canvasInput y mostraremos el canvasPuntuacion para que el jugador pueda ver los Records. Llamaremos a la funcion actualizarBD()
